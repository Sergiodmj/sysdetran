## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## Instalar o Botstrab

npm i bootstrap@5.3.3

## Instalar o Firebase

npm install firebase

## Instalar o firebase tool

 npm install -g firebase-tools

## Iniciar o firebase
 firebase init

## Fazer o deploy
 firebase deploy
